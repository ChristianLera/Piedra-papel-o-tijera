# 🪨 Piedra, Papel o Tijera - Versión Profesional

**Autor:** Christian Lera

Juego clásico de Piedra, Papel o Tijera implementado en Python con interfaz por consola, sistema de puntuación, múltiples rondas y validaciones completas.

## 🎮 Características

- ✅ Validación de entrada del usuario
- 🎲 Opción aleatoria para la computadora
- 🏆 Sistema de puntuación acumulativa
- 🔁 Juego por rondas ilimitadas
- 🎨 Emojis y diseño visual en consola
- 🧹 Limpieza automática de pantalla (Windows/Linux/Mac)
- 📊 Resumen final con ganador absoluto

## 🖥️ Requisitos

- Python 3.6 o superior
- No requiere bibliotecas externas (solo biblioteca estándar)

## 🚀 Ejecución

### Windows (con doble clic o terminal)
```batch
ejecutar.bat
```

### Windows (PowerShell)
```powershell
.\ejecutar.ps1
```

### Linux / macOS / Manual
```bash
python PiedraPapelTijera.py
```

## 🎯 Cómo jugar

1. Ejecuta el programa
2. En cada ronda escribe: `piedra`, `papel` o `tijera`
3. La computadora elegirá su opción automáticamente
4. Se mostrará el resultado de la ronda y el marcador
5. Decide si quieres seguir jugando (`s`) o salir (`n`)
6. Al final verás el resultado global
